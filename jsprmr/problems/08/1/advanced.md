# 補正点つき成績カード

## 問題

`input` には、合格ラインと、複数のチームの成績データが入っています。

各チームの成績データには、次のプロパティがあります。

- `name`: チーム名です。このプロパティがない場合があります。
- `base`: 基本点です。
- `adjustments`: 加点または減点を表す数値の配列です。

各チームについて、次のオブジェクトを作ってください。

- `name`: チーム名です。`name` プロパティがない場合だけ、文字列 `"未設定"` にしてください。`name` が空文字列 `""` や `null` の場合は、その値をそのまま使ってください。
- `total`: `base` と `adjustments` のすべての数値を足した数値です。
- `passed`: `total` が `input.line` 以上なら `true`、そうでなければ `false` です。

すべてのチームについてこのオブジェクトを作り、同じ順番の配列にしてください。

この課題では、チーム名を決める関数ではデフォルト引数を使い、合計点を計算する関数では Rest parameters を使ってください。`adjustments` を合計点の関数へ渡すときは、関数呼び出しの Spread 構文を使ってください。

## 入力

`input` には、次の形のオブジェクトが入っています。

```js
{
  line: 合格ラインの数値,
  records: チームの成績データの配列
}
```

## 出力

`output` に、各チームの成績カードを表すオブジェクトの配列を代入してください。

## 例

`input`:

```json
{
  "line": 70,
  "records": [
    { "name": "A班", "base": 50, "adjustments": [10, 5] },
    { "base": 62, "adjustments": [8] },
    { "name": "C班", "base": 80, "adjustments": [-15, 5] }
  ]
}
```

期待される `output`:

```json
[
  { "name": "A班", "total": 65, "passed": false },
  { "name": "未設定", "total": 70, "passed": true },
  { "name": "C班", "total": 70, "passed": true }
]
```
