# 1個ならそのままの合計金額

## 問題

`input` には、商品の単価と個数を表すオブジェクトが入っています。

`input.price` は商品の単価を表す数値です。
`input.count` は商品の個数を表す数値ですが、入っていない場合があります。

関数宣言で `totalPrice` という関数を定義してください。
この関数は単価 `price` と個数 `count` を受け取り、合計金額を数値で返します。
ただし、`count` が渡されなかった場合は、デフォルト引数を使って個数を `1` として扱ってください。

そのうえで、`input.price` と `input.count` を使って `totalPrice` を呼び出し、返り値を `output` に代入してください。

## 入力

`input` には次の形のオブジェクトが入っています。

```json
{
  "price": 120,
  "count": 3
}
```

`count` プロパティがない場合もあります。

## 出力

`output` に、合計金額を表す数値を代入してください。

## 例

`input`:

```json
{
  "price": 120,
  "count": 3
}
```

期待される `output`:

```json
360
```
