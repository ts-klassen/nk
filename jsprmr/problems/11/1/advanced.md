# 閉店までの売上集計

## 問題

`input` には、売上記録を先頭から順番に並べた数値の配列が入っています。

各数値の意味は次のとおりです。

- 正の数: 1件の売上金額
- 負の数: 取り消された記録なので集計しない
- `0`: 閉店を表す記録

先頭から順番に調べ、最初に `0` が出た時点で処理を止めてください。`0` より後ろの値はすべて無視します。

`0` が出る前にある正の数だけを集計し、`output` に次のプロパティを持つオブジェクトを代入してください。

- `count`: 集計した売上件数を表す数値
- `total`: 集計した売上金額の合計を表す数値
- `hasLarge`: `1000` 以上の売上が1件でもあれば `true`、なければ `false`

正の数が1つも集計されない場合、`count` は `0`、`total` は `0`、`hasLarge` は `false` にしてください。

## 入力

`input` には数値の配列が入っています。配列は空の場合もあります。

## 出力

`output` に、`count`、`total`、`hasLarge` を持つオブジェクトを代入してください。

## 例

`input`:

```json
[480, -20, 1200, 0, 3000]
```

期待される `output`:

```json
{
  "count": 2,
  "total": 1680,
  "hasLarge": true
}
```
