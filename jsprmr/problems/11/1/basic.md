# 数値を順番に合計する

## 問題

`input` に入っている配列の数値を、先頭から順番に合計してください。

この課題では、`for` 文と `input.length`、インデックスアクセスを使って配列の各要素を処理してください。

合計した数値を `output` に代入してください。

## 入力

`input` には数値の配列が入っています。

## 出力

`output` には、配列の数値をすべて合計した数値を代入してください。

`input` が空配列の場合は、`0` を代入してください。

## 例

`input`:

```json
[2, 4, 6]
```

期待される `output`:

```json
12
```
