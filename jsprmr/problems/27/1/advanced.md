# 整数で使える範囲

## 問題

実数の候補値から、整数として使える範囲を求めます。

`input.values` には、1 個以上の数値が入った配列が入っています。この配列の最小値と最大値を両端とする範囲を考えます。

さらに、整数の下限 `input.min` と上限 `input.max` が与えられます。`input.min` と `input.max` は整数で、どちらも範囲に含みます。

`input.values` から作った範囲と、`input.min` 以上 `input.max` 以下の範囲の両方に含まれる整数について、次のように `output` に配列を代入してください。

- 使える整数がある場合: `[最小の整数, 最大の整数]`
- 使える整数が 1 つだけの場合: `[その整数, その整数]`
- 使える整数がない場合: `[]`

`Math.min`、`Math.max`、`Math.ceil`、`Math.floor` を組み合わせて考えてください。

## 入力

`input` には、次の形のオブジェクトが入っています。

```js
{
  values: 数値の配列,
  min: 整数の下限,
  max: 整数の上限
}
```

## 出力

`output` に、使える整数の最小値と最大値を表す配列を代入してください。

使える整数がない場合は、空配列 `[]` を代入してください。

## 例

`input`:

```json
{
  "values": [2.2, 8.9, 5.5],
  "min": 3,
  "max": 8
}
```

期待される `output`:

```json
[3, 8]
```

`input.values` の範囲は 2.2 以上 8.9 以下です。この範囲に含まれる整数は 3 から 8 までで、`input.min` と `input.max` の範囲にも収まっています。

`input`:

```json
{
  "values": [1.2, 1.8],
  "min": 0,
  "max": 10
}
```

期待される `output`:

```json
[]
```

1.2 以上 1.8 以下に含まれる整数はないため、空配列になります。
