# 小数点以下を切り下げる

## 問題

`Math.floor` を使って、与えられた数値の小数点以下を切り下げてください。

`Math.floor(number)` は、`number` 以下で最大の整数を返します。負の数では、0 に近づけるのではなく、より小さい整数になることに注意してください。

## 入力

`input` には数値が入っています。

## 出力

`output` に、`Math.floor(input)` の結果である数値を代入してください。

## 例

`input`:

```json
3.8
```

期待される `output`:

```json
3
```
