# 独立した連番メーカー

## 問題

複数の名前付き連番メーカーを作り、呼び出された順に値を記録してください。

各連番メーカーは、作成時に `start` と `step` を受け取ります。

- 最初に呼び出されたときは `start` を返す
- 2 回目以降は、前回返した値に `step` を足した値を返す
- それぞれの連番メーカーは、ほかの連番メーカーとは独立した状態を持つ

外側の関数で現在値を表す変数を作り、その変数を参照する内側の関数を返すことで、クロージャーを使って連番メーカーを作ってください。

## 入力

`input` には、次の形のオブジェクトが入っています。

```json
{
  "settings": [
    { "name": "red", "start": 3, "step": 2 },
    { "name": "blue", "start": 10, "step": -1 }
  ],
  "calls": ["red", "blue", "red", "red", "blue"]
}
```

- `settings` は連番メーカーの設定を表す配列です。
- `settings` の各要素には、重複しない `name`、最初の値 `start`、増減値 `step` が入っています。
- `calls` は、呼び出す連番メーカーの `name` を順番に並べた配列です。
- `calls` に入っている `name` は、必ず `settings` に存在します。
- `calls` は空配列になることがあります。

## 出力

`output` に、`calls` の順番で各連番メーカーを呼び出した結果を並べた数値の配列を代入してください。

## 例

`input`:

```json
{
  "settings": [
    { "name": "red", "start": 3, "step": 2 },
    { "name": "blue", "start": 10, "step": -1 }
  ],
  "calls": ["red", "blue", "red", "red", "blue"]
}
```

期待される `output`:

```json
[3, 10, 5, 7, 9]
```
