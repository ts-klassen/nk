# クロージャーで回数を進める

## 問題

`input` には、数値 `start` と数値 `times` を持つオブジェクトが入っています。

引数 `start` を受け取り、呼び出されるたびに内部の数値を `1` ずつ増やして返す関数を返す `makeCounter` を作ってください。

`makeCounter(input.start)` でカウンターを 1 つ作り、そのカウンターを `input.times` 回呼び出してください。各呼び出しで返った数値を順に配列へ入れ、その配列を `output` に代入してください。

カウンターの現在値は、返された関数が外側の関数スコープの変数を参照し続けることで保持してください。

## 入力

`input` には次の形のオブジェクトが入っています。

```json
{
  "start": 3,
  "times": 4
}
```

- `start` はカウンターの初期値を表す数値です。
- `times` はカウンターを呼び出す回数を表す数値です。

## 出力

`output` に、カウンターを呼び出した結果を順に並べた数値の配列を代入してください。

## 例

`input`:

```json
{
  "start": 3,
  "times": 4
}
```

期待される `output`:

```json
[4, 5, 6, 7]
```
