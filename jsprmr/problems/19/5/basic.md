# 商品オブジェクトの説明メソッド

## 問題

`input` には、商品の名前と価格を表すオブジェクトが入っています。

`name` と `price` をプロパティに持つオブジェクトを作り、そのオブジェクトに説明文を返すメソッドを定義してください。

メソッドの中では `this.name` と `this.price` を使って、次の形式の文字列を返してください。

```text
商品名は価格円です
```

メソッドは Arrow Function ではなく、通常の関数またはメソッドの短縮記法で定義してください。

最後に、そのメソッドを `オブジェクト名.メソッド名()` の形で呼び出し、戻り値を `output` に代入してください。

## 入力

`input` には、次の形のオブジェクトが入っています。

```js
{
  name: 商品名を表す文字列,
  price: 価格を表す数値
}
```

## 出力

`output` に、商品の説明文を表す文字列を代入してください。

## 例

`input`:

```json
{
  "name": "ノート",
  "price": 120
}
```

期待される `output`:

```json
"ノートは120円です"
```
