# エラーメッセージを受け取る

## 問題

`input` には、エラーメッセージとして使う文字列が入っています。

`try...catch` を使って、`try` ブロックの中で `new Error(input)` を `throw` してください。

`catch` 節では、受け取ったエラーオブジェクトの `message` プロパティを読み取り、その文字列を `output` に代入してください。

## 入力

`input` には文字列が入っています。

## 出力

`output` に、`catch` 節で受け取った `Error` オブジェクトの `message` プロパティの値を文字列として代入してください。

## 例

`input`:

```json
"名前が空です"
```

期待される `output`:

```json
"名前が空です"
```
