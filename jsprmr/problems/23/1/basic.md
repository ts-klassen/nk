# バッジ番号の種類

## 問題

展示会の受付では、来場者のバッジ番号をスキャンした順に記録しています。
同じ来場者が何度も通ることがあるため、同じバッジ番号が複数回入ることがあります。

`Set` を使って、記録に含まれるバッジ番号が何種類あるかを求めてください。

## 入力

`input` には、バッジ番号を表す文字列の配列が入っています。

## 出力

`output` に、バッジ番号の種類数を数値として代入してください。

## 例

`input`:

```json
["A12", "B07", "A12", "C30"]
```

期待される `output`:

```json
3
```
