# 部屋ごとの参加者数

## 問題

`input` には、入退室の記録を表すオブジェクトの配列が入っています。

各記録は `{ "room": 部屋名, "user": 利用者名, "action": 操作 }` の形です。`room` と `user` は文字列で、`action` は `"enter"` または `"leave"` です。

記録を先頭から順に処理し、最終的に入室中の利用者が 1 人以上いる部屋について、部屋名と人数を求めてください。

次のルールで処理します。

- 同じ部屋に同じ利用者が何度 `"enter"` しても、その部屋では 1 人として扱う
- 入室していない利用者の `"leave"` は何も変更しない
- 存在しない部屋に対する `"leave"` は何も変更しない
- `"leave"` によって部屋の人数が 0 人になったら、その部屋を取り除く
- 取り除かれた部屋にあとで誰かが `"enter"` した場合、その時点で新しく追加された部屋として扱う

部屋の並び順は、現在残っている部屋が追加された順にしてください。

`Map` と `Set` を使って、部屋ごとの参加者集合を管理してください。

## 入力

`input` には入退室記録の配列が入っています。

## 出力

`output` に、最終的に入室中の利用者が 1 人以上いる部屋を表す配列を代入してください。

各要素は `[部屋名, 人数]` の配列にしてください。部屋名は文字列、人数は数値です。残っている部屋がない場合は、空配列 `[]` を代入してください。

## 例

`input`:

```json
[
  { "room": "red", "user": "A", "action": "enter" },
  { "room": "red", "user": "B", "action": "enter" },
  { "room": "blue", "user": "A", "action": "enter" },
  { "room": "red", "user": "A", "action": "enter" },
  { "room": "red", "user": "A", "action": "leave" },
  { "room": "green", "user": "D", "action": "enter" },
  { "room": "blue", "user": "C", "action": "leave" },
  { "room": "red", "user": "C", "action": "enter" }
]
```

期待される `output`:

```json
[["red", 2], ["blue", 1], ["green", 1]]
```
