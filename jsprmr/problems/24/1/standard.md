# 歩幅つきの数列ジェネレータ

## 問題

`input.start` から始めて、次の値が前の値に `input.step` を足した値になる数列を考えます。

`function*` と `yield` を使って、この数列を `input.count` 個だけ生成するジェネレータ関数を定義してください。作った Generator オブジェクトを `for...of` で反復し、取り出した値を順番に配列へ入れてください。

`input.count` が `0` の場合、配列は空にしてください。

## 入力

`input` には次のプロパティを持つオブジェクトが入っています。

- `start`: 最初の数値
- `step`: 次の値へ進むときに足す数値
- `count`: 生成する値の個数を表す `0` 以上の整数

## 出力

生成した数値を順番に入れた配列を `output` に代入してください。

## 例

`input`:

```json
{
  "start": 3,
  "step": 2,
  "count": 4
}
```

期待される `output`:

```json
[3, 5, 7, 9]
```
