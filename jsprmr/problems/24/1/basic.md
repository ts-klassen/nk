# yieldで値を取り出す

## 問題

`input` は 2 つの要素を持つ配列です。

`function*` でジェネレータ関数を定義し、`input[0]`、`input[1]` の順に `yield` してください。

そのジェネレータ関数を呼び出して Generator オブジェクトを作り、`next` メソッドを 2 回呼び出してください。

2 回の `next` メソッドの戻り値を、呼び出した順番の配列にして `output` に代入してください。

## 入力

`input` には、2 つの文字列または数値を持つ配列が入っています。

## 出力

`output` には、2 回の `next` メソッドの戻り値を入れた配列を代入してください。

配列の各要素は、`{ value: 値, done: false }` という形のオブジェクトにしてください。

## 例

`input`:

```json
["赤", "青"]
```

期待される `output`:

```json
[
  { "value": "赤", "done": false },
  { "value": "青", "done": false }
]
```
