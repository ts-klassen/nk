# 変数名を評価した結果

## 問題

JavaScriptでは、変数名を評価すると、その変数に入っている値が評価結果になります。

`input` を評価した結果と同じ値を、`output` に代入してください。

## 入力

`input` には、数値または文字列が入っています。

## 出力

`output` に、`input` を評価した結果と同じ値を代入してください。

`output` の型は、`input` と同じ型にしてください。

## 例

`input`:

```json
"book"
```

期待される `output`:

```json
"book"
```
