# 名前つきインポート文を組み立てる

## 問題

ECMAScriptモジュールでは、別のモジュールが名前つきエクスポートした値を、名前を指定してインポートできます。

`input` には、インポートする名前 `name` と、インポート元の相対URL `path` を持つオブジェクトが入っています。

`name` と `path` を使って、次の形のインポート文を文字列として組み立ててください。

```js
import { name } from "path";
```

出力する文字列では、インポート元をダブルクォートで囲み、文末にセミコロンを付けてください。

## 入力

`input` には次の形のオブジェクトが入っています。

```js
{
  name: "インポートする名前",
  path: "インポート元の相対URL"
}
```

`name` は名前つきエクスポートされた名前として使える文字列です。
`path` は `./` または `../` で始まる相対URLの文字列です。

## 出力

`output` に、名前つきインポート文を表す文字列を代入してください。

## 例

`input`:

```json
{
  "name": "sum",
  "path": "./math.js"
}
```

期待される `output`:

```json
"import { sum } from \"./math.js\";"
```
