# Stringラッパーから値を取り出す

## 問題

`input` には、`new String(...)` で作られた `String` ラッパーオブジェクトが入っています。

`valueOf` メソッドを呼び出して、ラッパーオブジェクトに包まれているプリミティブ型の文字列を取り出してください。

取り出した文字列を `output` に代入してください。

## 入力

`input` には `String` ラッパーオブジェクトが入っています。

## 出力

`output` にプリミティブ型の文字列を代入してください。

`String` ラッパーオブジェクトのまま代入しないでください。

## 例

`input`:

```js
new String("apple")
```

期待される `output`:

```json
"apple"
```
