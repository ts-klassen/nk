# テンプレートでラベルを作る

## 問題

`input` に入っている商品名と個数を使って、次の形式の文字列を作成してください。

```text
商品: 商品名 / 個数: 個数
```

テンプレートリテラルの `${}` を使って文字列を組み立ててください。

## 入力

`input` には、次のプロパティを持つオブジェクトが入っています。

- `name`: 商品名を表す文字列
- `count`: 個数を表す数値

## 出力

`output` には、指定された形式の文字列を代入してください。

## 例

`input`:

```json
{
  "name": "ペン",
  "count": 3
}
```

期待される `output`:

```json
"商品: ペン / 個数: 3"
```
