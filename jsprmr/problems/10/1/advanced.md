# うるう年を判定する

## 問題

`input` に西暦の年を表す数値が入っています。

次のルールにしたがって、うるう年なら `output` に真偽値 `true` を、うるう年でなければ真偽値 `false` を代入してください。

- `400` で割り切れる年は、うるう年
- `400` で割り切れず、`100` で割り切れる年は、うるう年ではない
- `100` で割り切れず、`4` で割り切れる年は、うるう年
- それ以外の年は、うるう年ではない

`Date` オブジェクトは使わず、剰余演算子 `%` と `if`、`else if`、`else` による条件分岐で判定してください。

## 入力

`input` には `1` 以上の整数が入っています。

## 出力

`output` には真偽値 `true` または `false` を代入してください。

## 例

`input`:

```json
2000
```

期待される `output`:

```json
true
```
