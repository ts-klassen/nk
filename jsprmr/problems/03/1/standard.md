# 表示名の更新

## 問題

`input` には、表示名として使う文字列が入っています。

まず、`input` の値を再代入しない変数として保存してください。
次に、表示名用の変数を文字列 `"ゲスト"` で用意してください。
その後、表示名用の変数へ、保存しておいた `input` の値を再代入してください。

最後に、表示名用の変数の値を `output` に代入してください。

## 入力

`input` には文字列が入っています。

## 出力

`output` に、最終的な表示名の文字列を代入してください。

## 例

`input`:

```json
"Sora"
```

期待される `output`:

```json
"Sora"
```
