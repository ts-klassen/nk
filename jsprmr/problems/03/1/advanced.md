# 札の入れ替え記録

## 問題

3 つの作業台に置かれた札を、変数で管理してください。

`input` には、最初に左の作業台へ置かれている札の文字列が入っています。

最初の状態は次の通りです。

- 左の作業台: `input` の値
- 中央の作業台: `"未記入"`
- 右の作業台: `"未記入"`

その後、次の順番で値を代入してください。

1. 中央の作業台へ、左の作業台の値を代入する
2. 左の作業台へ、`"空"` を代入する
3. 右の作業台へ、中央の作業台の値を代入する
4. 中央の作業台へ、左の作業台の値を代入する
5. 左の作業台へ、`"完了"` を代入する

最後に、右の作業台に入っている文字列を `output` に代入してください。

再代入する変数には `let` を使い、再代入しない値には `const` を使ってください。文字列の連結や計算は必要ありません。

## 入力

`input` には文字列が入っています。

## 出力

`output` に、最後に右の作業台へ入っている文字列を代入してください。

`output` の値は文字列にしてください。

## 例

`input`:

```json
"赤い札"
```

期待される `output`:

```json
"赤い札"
```
