# ECMAScriptの役割を選ぶ

## 問題

`input` には、ECMAScriptに関係する用語が文字列で入っています。

この単元で学んだそれぞれの役割を思い出し、用語に対応する役割ラベルを `output` に代入してください。

役割ラベルは、文字列 `"仕様"`、`"標準化団体"`、`"技術委員会"` のいずれかです。

## 入力

`input` には文字列 `"ECMAScript"`、`"Ecma International"`、`"TC39"` のいずれかが入っています。

## 出力

`output` には、`input` の用語に対応する役割ラベルを文字列で代入してください。

## 例

`input`:

```json
"ECMAScript"
```

期待される `output`:

```json
"仕様"
```
