# 動的なキーで設定値を読む

## 問題

`input` には、`settings` プロパティと `key` プロパティを持つオブジェクトが入っています。

`input.settings` は設定値を持つオブジェクトです。`input.key` は、取り出したいプロパティ名を表す文字列です。

`input.settings` が `input.key` と同じ名前のプロパティを持つ場合は、そのプロパティ値を `output` に代入してください。

持たない場合は、文字列 `"未設定"` を `output` に代入してください。

## 入力

`input` にはオブジェクトが入っています。

`input.settings` にはオブジェクト、`input.key` には文字列が入っています。

設定値には、文字列、数値、真偽値が入ります。

## 出力

`output` には、指定されたプロパティが存在する場合はそのプロパティ値を代入してください。

指定されたプロパティが存在しない場合は、文字列 `"未設定"` を代入してください。

## 例

`input`:

```json
{
  "settings": {
    "theme": "dark",
    "fontSize": 16
  },
  "key": "theme"
}
```

期待される `output`:

```json
"dark"
```
