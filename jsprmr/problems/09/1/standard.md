# 関数宣言文と関数式

## 問題

`input` には数値が入っています。

次の 2 つの関数を作り、関数呼び出し式の評価値を使って `output` に数値を代入してください。

- `triple` は、受け取った数値を 3 倍して返す関数宣言文として作る
- `addSeven` は、受け取った数値に 7 を足して返す関数式を `const` 変数へ代入して作る

`output` には、`input` を `triple` に渡した評価値を、さらに `addSeven` に渡した結果を代入してください。

関数宣言文はブロックで終わる文なので末尾のセミコロンは不要です。関数式を代入する `const` の変数宣言文と、`output` への代入文の末尾にはセミコロンを付けてください。

## 入力

`input` には数値が入っています。

## 出力

`output` に数値を代入してください。

## 例

`input`:

```json
4
```

期待される `output`:

```json
19
```
