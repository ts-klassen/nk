# 整数スコアの受付判定

## 問題

提出フォームから受け取ったスコア候補を、採点対象として受け付けてよいか判定します。

`input` は、文字列、数値、真偽値、`null` などの値です。

次の条件をすべて満たす場合だけ、受け付けてよい値とします。

- `input` の型が String 型または Number 型である
- `input` を `Number(input)` で数値に変換した結果が `NaN` ではない
- `input` を `String(input)` で文字列にし、`Number.parseInt(..., 10)` で変換した値が、`Number(input)` の結果と厳密に等しい
- 数値に変換した結果が `0` 以上 `100` 以下である

条件を判定するときは、`==` ではなく `===` を使ってください。

## 入力

`input` には、スコア候補として受け取った値が入っています。

## 出力

`output` に、受け付けてよい場合は真偽値の `true`、そうでない場合は真偽値の `false` を代入してください。

## 例

`input`:

```json
"80"
```

期待される `output`:

```json
true
```
