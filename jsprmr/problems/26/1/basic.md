# 時刻値を取り出す

## 問題

`input` には `Date` インスタンスが入っています。

`Date` インスタンスの `getTime` メソッドを使い、そのインスタンスが持つ時刻値を `output` に代入してください。

## 入力

`input` には有効な `Date` インスタンスが入っています。

## 出力

`output` には、`input` が持つ時刻値を数値で代入してください。

## 例

`input`:

```js
new Date("1970-01-01T00:00:01.000Z")
```

期待される `output`:

```json
1000
```
