# 加算後のISO文字列

## 問題

開始時刻と、そこから何分後かを表す数値が与えられます。

開始時刻から指定された分数だけ後の時刻を、UTCのISO 8601形式の文字列にしてください。

## 入力

`input` には、次のプロパティを持つオブジェクトが入っています。

- `startedAt`: 末尾に `Z` が付いたISO 8601形式の日時文字列
- `minutes`: 何分後かを表す0以上の整数

## 出力

`output` に、加算後の時刻を表す文字列を代入してください。

文字列は `Date` インスタンスの `toISOString` メソッドで得られる形式にしてください。

## 例

`input`:

```json
{
  "startedAt": "2024-05-01T09:00:00.000Z",
  "minutes": 30
}
```

期待される `output`:

```json
"2024-05-01T09:30:00.000Z"
```
