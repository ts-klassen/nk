# 参照先を選ぶ

## 問題

調べたい内容の種類に応じて、最初に確認する参照先を選びます。

`input` には、調べたい内容を表す文字列が入っています。次のルールで、対応する参照先の文字列を `output` に代入してください。

- `input` が `"JavaScript API"` または `"DOM API"` の場合は、`"MDN Web Docs"`
- `input` が `"Node.js API"` の場合は、`"Node.js Documentation"`
- `input` が `"ライブラリ"` または `"ツール"` の場合は、`"公式サイトまたはリポジトリ"`
- それ以外の場合は、`"公式情報を確認"`

## 入力

`input` には文字列が入っています。

## 出力

`output` に参照先を表す文字列を代入してください。

## 例

`input`:

```json
"DOM API"
```

期待される `output`:

```json
"MDN Web Docs"
```
