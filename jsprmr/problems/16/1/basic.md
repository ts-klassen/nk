# 先頭のCode Point

## 問題

`input` に入っている文字列の先頭にある文字を、Code Point として扱ってください。

`String.prototype.codePointAt` を使い、インデックス `0` にある文字の Code Point を数値として取得してください。

取得した数値を `output` に代入してください。

## 入力

`input` には空ではない文字列が入っています。

## 出力

`output` には、`input` の先頭にある文字の Code Point を数値で代入してください。

## 例

`input`:

```json
"A"
```

期待される `output`:

```json
65
```
