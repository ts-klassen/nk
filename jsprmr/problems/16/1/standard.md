# コードポイントの16進数リスト

## 問題

`input` には文字列が入っています。

`input` に含まれる文字を Code Point 単位で先頭から順に調べ、それぞれの Code Point を 16 進数の文字列に変換してください。

変換した文字列を順に入れた配列を、`output` に代入してください。

16 進数の文字列には `0x` や `U+` を付けず、`Number.prototype.toString(16)` で得られる小文字の文字列にしてください。

`input` が空文字列の場合は、空の配列を `output` に代入してください。

## 入力

`input` には文字列が入っています。

## 出力

`output` に、Code Point ごとの 16 進数文字列を並べた配列を代入してください。

## 例

`input`:

```json
"A🍣あ"
```

期待される `output`:

```json
["41", "1f363", "3042"]
```
