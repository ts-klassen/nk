# typeofでデータ型を調べる

## 問題

`input` に入っている値へ `typeof` 演算子を使い、その評価結果を `output` に代入してください。

`typeof` の評価結果は文字列です。たとえば、数値に `typeof` を使うと `"number"` になります。

## 入力

`input` には、真偽値、数値、文字列、`null`、配列、オブジェクトのいずれかの値が入っています。

## 出力

`output` に、`typeof input` の評価結果である文字列を代入してください。

## 例

`input`:

```json
42
```

期待される `output`:

```json
"number"
```
