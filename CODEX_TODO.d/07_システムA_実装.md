# システム A を実装する

## 目的

教材で扱う完成実装として、書籍管理システムを実装する。

## 実装対象

- Express アプリ
- TypeScript 設定
- MySQL 接続
- DB 初期化手順
- 共通エラーレスポンス
- 入力バリデーション
- Basic 認証
- Argon2 によるパスワードハッシュ化
- 書籍 CRUD
- 読書メモ CRUD

## 方針

- 過度な抽象化はしない。
- フレームワーク固有の高度な設計に寄せない。
- 受講生が読み切れるファイル数にする。
- SQL は直接書く。
- フロントエンドとの型共有はしない。
- 読書メモは所有者だけが閲覧・更新・削除できる。
