# システム A の教材を書く

## 目的

書籍管理システムを題材に、バックエンド開発の基礎を学ぶ教材を作成する。

## 教材で扱う内容

- REST API の基本
- HTTP メソッドとステータスコード
- JSON リクエストとレスポンス
- Express v5 の最小構成
- TypeScript と `tsc` ビルド
- MySQL 接続
- SQL を直接実行する実装
- 入力バリデーション
- 共通エラーレスポンス
- Basic 認証
- Argon2 によるパスワードハッシュ化
- 認証済みユーザーだけが扱えるリソース

## 教材で深入りしない内容

- Node.js の先端機能
- Express 固有の高度な設計
- ORM
- migration ツール
- JWT
- セッション管理
- JOIN など複雑な SQL
- フロントエンドとの型共有
